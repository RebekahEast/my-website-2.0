My Projects

Learn about my projects here

I've been working with a range of coding languages since June 2022. In this time, I've kept practising, learning and developing my skills! Below are a few of my projects, have a look through and see what you think. I keep pushing myself to take on bigger and better projects.



Hobby Page

Using my experience with HTML and CSS, I created a webpage of one of my interests - plant care! The idea for this was to make the page as user friendly as possible for a new time plant owner, or someone looking to care for their plants.

Project Page

I created a webpage to host all my projects I have been working on. It allows the user to toggle the text and read more about each one if they'd like - it also has a link to each project via the title. The page was created with a flexbox.

Movie Data

I created a movie directory to show some top movies, and also allow the user to input data about their best movie choices too. The webpage has a submit option for sending in own responses, and the data will appear underneath in a flexbox.

Cow Page

I wanted to test my flexbox and javascript learnings, so I created a cow page where there are different cow breeds and facts to learn about each of them! If you hover over each cow, then you'll get a fact about that breed courtesy of Javascript.

WIP: 1024

This project is a work in progress. I am trying to recreate a beloved game of mine (1024). A lot of the code sits in Javascript, and I'm pushing myself to create this really big project in order to learn more about the language.
